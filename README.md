# LA-Tech-Rotten-Tomatoes-Clone

## Overview
This project is a cross-platform mobile app (iOS and Android) that allows users to:
- Browse a list of movies and TV shows  
- Vote with a thumbs up or thumbs down  
- View aggregated voting results  

The primary goal of this project is to gain hands-on experience with modern development technologies, including mobile development, backend systems, and collaborative workflows—not to build a production-ready app.

---

## Project Goals
- Practice cross-platform mobile development  
- Design and build a backend API  
- Explore modern tooling and workflows  
- Collaborate across multiple subteams  

---

## Team Structure

### Product & UI
- Define and prioritize app requirements  
- Create UI mockups and user flows  

### Infrastructure
- Set up and manage project tools:
  - Source control (GitHub)  
  - Communication (Slack/Discord)  
  - Scheduling (Google Calendar)  
- Research and introduce development tools (e.g., vibe coding tools)  

### Frontend
- Choose a mobile framework (e.g., Flutter or React Native)  
- Build and test the mobile app  
- Implement UI based on product requirements  
- Integrate with backend APIs  

### Backend
- Design data schema and API  
- Implement features:
  - Movie/TV show list  
  - Voting (thumbs up/down)  
  - User authentication  
- Deploy and manage backend infrastructure
